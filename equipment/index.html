裝備

在腐朽的墳墓與被世界遺忘的財寶堆中堆滿了許許多多有用的物品。戰士可以在其中找到一把嶄新的鋒利寶劍、而盜賊也許能從中偶然找到一瓶致命毒藥。大部分的物品都是尋常無奇──沒有魔法、也沒什麼真正獨特的特質。對於遊戲機制的動作來說，任何帶有魔法或獨樹一格的物品都會被視作凡物。此外，戰士的招牌武器也永遠不會被視作凡物。

每件裝備都會帶有複數個標籤。這些標籤將告訴你這件裝備會如何影響使用它的角色（像是 +護甲值），或暗示它該如何被使用（例如範圍標籤）。就如同《地城世界》中的其他事物，它們能夠引導你在遊戲中創造故事情節。比如說，如果一把武器帶有「不便」標籤，那麼這可能意味著當你在劈砍檢定中失敗時，這把武器會有很大的機會脫手，落到地面。

以下列出的絕非詳盡完整的清單──自由地創造你自己的標籤吧。

泛用裝備標籤

這些泛用標籤可以適用於幾乎任何裝備。你會在防具、武器、和一般冒險工具上看到它們。

內服 (Applied): 它只有在你確切地讓目標服用、或混進他們嚥下的飲食內才會生效。

不便 (Awkward): 它相當笨重且難以使用。

+修正 (+Bonus): 它將改變你在特定情況下的行動成效。比如說「憶述見聞下次+1」或是「砍劈持續-1」等等。

n 金幣 (n coins): 它在正常情況下值多少錢。如果價格包括「－魅力」的文字，則將其價格減去購買者的魅力值（而非修正值）。

危險 (Dangerous): 使用它很容易讓你惹上麻煩。如果你與它互動時沒有準備適當的預防措施，則GM可以自由地描述你愚蠢行動的後果。

食物 (Ration): 它是可食用的，或多或少。

限定 (Requires): 它只對某些人有用。如果你無法滿足它的條件，則它的效果會減弱，或甚至完全無效。

緩慢 (Slow): 使用它需要花費數分鐘或更久的時間。

接觸 (Touch): 它需要透過直接接觸到目標肌膚的方式來被使用。

雙手 (Two-handed): 它需要兩隻手才能被有效地使用。

n 重量 (n weight): 計算列有重量的物品的總重，並與你的負載量相比較。未列有重量的東西不能被隨身攜帶。100枚標準面額的金幣為1 重量。有著相同價值的寶石或美術品可能會更輕或更重。

穿戴 (Worn): 你必須要穿戴它才能夠使用它。

n 使用次數 (n Uses): 它只能被使用 n 次。

武器 (Weapons)

殺死怪物的並非武器，而是揮舞它們的人。這也是為什麼《地城世界》中的武器並沒有列出其傷害數據。武器的主要重點在於其標籤，用來描述這把武器的用途與特性。一把匕首的重點並不在於它能造成比其他刀劍更多或更少的傷害，而在於它的小巧靈活，且能輕易地在極近距離下重創對手。因此，法師手中匕首的威脅性，就很明顯不如技巧純熟的戰士手上的那把匕首。

武器標籤

武器可能會帶有一些主要用以幫助你描述它們的標籤（例如「生鏽」或「閃閃發光」），但以下這些標籤除了描述的用途之外，還有著其他遊戲機制上的效果。

n 彈藥 (n Ammo): 它被視作適當遠程武器的彈藥。數字ｎ並不代表確切的箭矢或投石數量，而是代表你手頭上還剩下什麼。

強力 (Forceful): 它可以迫使某人後退幾步，或甚至讓他們失足倒地。

+n 傷害 (+n Damage): 它對你的敵人而言特別危險。當你造成傷害時，你使傷害增加ｎ點。

無視護甲 (Ignores Armor): 造成的傷害並不需要減去對方的護甲值。

毀亂 (Messy): 它以特別毀滅性的方式造成傷害，將人或物撕成碎片。

n 穿透(n Piercing): 它能夠穿透護甲。當你以ｎ穿透造成傷害時，對該攻擊而言敵人的護甲將減少ｎ點。

靈巧 (Precise): 它有助於謹慎地施加打擊。你以此武器進行砍劈動作時，使用敏捷進行檢定，而非力量。

裝填 (Reload): 在你以它進行攻擊之後，會需要稍多一點的時間來重置裝填才能進行下一次攻擊。

暈眩 (Stun): 當你以它進行攻擊時，它將造成暈眩傷害，而非正常的傷害。

投擲 (Thrown): 把它擲向目標以造成傷害。若你以此武器進行射擊動作，你在擲出7~9時將不能選擇標記彈藥-1；一旦你把它擲出，直到你把它拾回之前都將視作失去它。

武器也帶有用來表示其作用範圍的標籤。地城世界並不會為「最佳作用範圍」而提供減值或加值，但如果你的武器帶著「觸及」標籤，而你的敵人距離你10碼之外，那麼玩家將會發現他很難說明自己是如何使用這把武器來對付那名敵人。

觸及 (Hand): 它適合用來攻擊位於你徒手接觸範圍內的事物，不能再遠了。

近身 (Close): 它適合用來攻擊位於徒手觸及距離再加上一到二英尺範圍內的事物。

及遠 (Reach): 它適合用來攻擊數英尺遠的事物－－或許最遠十英尺。

短程 (Near): 它適合用來攻擊你能夠看到其眼白的目標。

遠程 (Far): 它適合用來攻擊位於吶喊所能傳達距離內的事物。

防具 (Armor)

防具相當沉重、難以穿脫、且穿起來天殺的難受。有些職業因為受過較好的訓練而能忽略這些缺點，但事實上任何人都能穿上這些防具，並享有其帶來的好處。

防具標籤

防具跟武器一樣帶有標籤。儘管有些標籤只是純粹的描述詞，但下列標籤能夠在玩家穿戴它們的時候帶來一些機制上的效果。

n 護甲 (n Armor): 它能夠保護你不受傷害。當你受到傷害時，將總傷害減去你的護甲值。如果你有超過一個帶有ｎ護甲標籤的物品，則只有數值最高的那個會納入計算。

+n 護甲 (+n Armor): 它能夠保護你，且能夠與其他防具的護甲值疊加。將它的數值加到你的總護甲值中。

笨重 (Clumsy): 穿戴它時將難以四處走動。當使用它時，檢定持續-1。這個減值會累積。

魔法物品

這個世界上存在著遠比刀劍革甲更加奇妙的事物。魔法物品，正是這類內部蘊含著神奇力量的非凡之物。

魔法物品是為了讓你創造屬於你自己的遊戲而存在的。玩家可以透過進行法師的儀式或其他類似動作來製造魔法物品。GM也可以將魔法物品做為戰利品或任務獎勵導入遊戲之中。下方的列表中僅僅是用來提供一些靈感，實際上魔法物品的效果與樣貌最終仍是由你來決定。

當製造你自己的魔法物品時，記住這些物品是充滿魔法的。單純的加減值，例如 +1 傷害，仍只是凡物的領域──魔法物品應該要能帶來更多更有趣的加成。

阿爾戈-坦恩, 神聖復仇者 (Argo-Thaan, Holy Avenger) 近身, 2 重量

儘管這個世界上有著種種寶劍，但阿爾戈-坦恩僅此一把。它是黃金、白銀、和光明之刃，被所有將善良視為真理的教團和宗教尊為聖物。它的觸碰是一種祝福，而對許多人來說，僅僅看見它就會流下喜悅的淚水。

當被握在聖武士的手中，它將能造成強而有力的打擊。持用它的聖騎士將使自己的傷害骰提升至 d12，且能夠使用所有的聖騎士動作。同時，阿爾戈-坦恩能夠傷害任何邪惡的生物，無論它披上了任何防護都一樣。沒有任何邪惡生物能夠觸碰阿爾戈-坦恩而不承受劇烈痛苦。在任何聖武士以外的人手中，它就僅僅是一把普通的長劍，且比大部分的長劍都來得更沉重累贅──它將因此獲得不便標籤。

儘管阿爾戈-坦恩並不具備智能，但它永遠會被真正的善良所吸引，就彷彿鋼鐵被磁鐵所吸引一樣。

黃泉之箭 (Arrows of Acheron) 1 彈藥, 1 重量

由一名盲眼製箭師在黑暗中所打造，這些箭矢能夠在最深沉的黑暗中找到它們的目標。弓箭手可以在失明、黑暗中、被厚重布料遮住雙眼時射出這些箭矢，而仍能保證精準命中目標。然而，倘若日光照到了這些箭矢上，它們將如陰影和塵土般消散。

征服王之斧 (Axe of the Conqueror-King) 近身, 1 重量

卡科薩尖槍 (The Carcosan Spire) 及遠, 投擲, 3 重量

若塗抹於門口的邊緣、或被用來畫出一個圓形區域，則這個膏油將會驅散任何原生於異位面的生物。牠們無法穿過被這個膏油圈起的區域邊緣。這個膏油能夠持續一整天，在這之後它便會被吸收或蒸發消散。

哀嘆遺物 (Lamented Memento) 0 重量

✴擲出失敗時，GM將會詢問你「你最害怕的是什麼？」。理所當然，你必須誠實地回答。

薩塔爾說書鴨鴨 (The Sartar Duck) 0 重量

一隻手工雕刻而成的奇怪木製鴨子。到底是誰會做出這種有趣的玩意兒？當你帶著它時，你將發現自己變成了一名天賦異稟的說書人──不管使用什麼語言，你都能讓任何聽眾清楚明白的理解你和你所述說的故事。即使他們聽不懂你的話，也能明白你的意思。

安娜莉絲的眼淚 (Tears of Annalise) 0 重量

拇指大小的濁紅色寶石，安娜莉絲的眼淚總是成對出現。當被兩個不同的人吞下後，這對寶石將會把這兩個人聯繫在一起──當其中一方感受到強烈的情緒（尤其是傷心、失落、恐懼、或渴望）時，另一方也會感受到相同的情緒。這個效果將持續直到其中一方讓另一方濺出鮮血。

傳送房間 (Teleportation Room) 緩慢

詹姆斯．九指，一名古怪的天才巫師，打造了這個房間大小的魔法裝置。傳送房間是一間石室，刻滿了散發著淡淡藍光的符文和草書。當你進入這個房間並大聲說出一個地點的名稱時，擲+智力：
✴擲出10+時，你確實抵達了你想要去的場所。
✴擲出7-9時，GM將選擇讓你抵達附近的一處安全場所。
✴擲出失敗時，你來到某個場所。也許就在附近？但肯定不怎麼安全。怪事經常會發生在那些嘗試使用這個裝置來扭曲時間和空間的人身上。

蒂蒙之甲 (Timunn’s Armor) 1 護甲, 1 重量

一套不顯眼的護甲，它在不同人的眼中將呈現出不同的樣貌，並會與合適的服裝融為一體。對任何凝視著穿戴者的人來說，穿戴者永遠都像是走在時尚尖端。

提多的誠實脂燭 (Titus’ Truthful Tallow) 0 重量

一個由象牙色與銅色獸脂和一根銀絲燈芯構成的蠟燭。當被點燃時，任何被它燭光照到的人都無法說出任何謊言。他們可以保持沉默或想辦法掩飾，但當被直接質問時，他們將只能說出真話。

妙戲繩 (Tricksy Rope) 1 重量

這條繩子會聽人話，也會表演各種把戲，就如同一條更聰明且更馴順的蛇一樣。命令它「盤繞」、「鬆垂」、或「繩子，過來」，然後它就會依言行事。

純銀之手 (The Sterling Hand) 0 重量

由矮人白鐵匠打造而成，這個表面光亮如鏡的金屬義手中深深刻印著力量與復興的符文。被設計來代替那些因礦坑事故而被重傷或摧毀的肢體，純銀之手能夠接上斷肢的缺口，無論新舊，並表現的強壯而結實。它可以被當作武器使用（觸及範圍），且因為它是以足夠純粹的銀打造，因此它也能夠用來傷害那些會被銀製品影響的生物。

維利烏斯的護手 (Vellius’s Gauntlets) 1 重量

以 笨拙的維利烏斯、奶油手維利烏斯、呆子維利烏斯 之名打造而成，這雙以單薄布料織成的手套能夠防止你失手放開任何你無意弄掉的東西。舉例來說，你不會被繳械、也不會失手從任何繩索或爬梯上摔落。不過要注意，若有什麼東西用力拉著你的雙腳，而你又緊緊抓著某個牢固的東西，那麼這雙手套就可能讓情況變得非常糟糕。

忤逆長刀 (Violation Glaive) 及遠, 2 重量

忤逆長刀是一把傳奇刃器，由奇怪的青綠鋼鐵鍛造而成，且傳說是來自於某個可怕的未來。這把刃器不僅會切開它所傷害之人的肉體，更會狠狠打擊目標的心靈。當你在進行砍劈動作時擲出 10+ 的結果時，你得到一個額外的選項：你可以向目標如常造成傷害，承受敵人的反擊，並將你所選擇的情緒灌注到目標的心中（可能是恐懼、崇敬、或信任）。

斬首劍 (Vorpal Sword) 近身, 3 穿透, 2 重量

劍聲嗖嗖，人頭落地。比世上任何東西都還要鋒利，這把看上去平凡無奇的長劍只有一個目的：把一個東西和另外一個東西斬斷分離──也許把肢體從軀幹上切開、或者把人們與他們的性命一刀兩斷。當你使用斬首劍來造成傷害時，你的敵人必須選擇某個事物（一個物品、一個優勢、或一個肢體）並永久地失去它。
